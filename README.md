# 📿 Japa Mala Counter

A Progressive Web App (PWA) for mantra chanting practice with voice recognition.

## Features

- 🗣️ **Voice Recognition** - Auto-detects Hindi mantra words (हरे, कृष्णा, राम)
- 🎤 **Audio Calibration** - Custom word detection with calibration
- 👆 **Tap Mode** - Full-screen tap counting (1 tap = 1 chant)
- 📊 **Statistics** - Track weekly progress and session history
- 🔥 **Streak Tracking** - Daily chanting streak
- 📱 **Works Offline** - Install on home screen like a native app
- 🌐 **Cross-Platform** - Works on iOS, Android, and Desktop

## Supported Mantras

- 🙏 Hare Krishna Maha Mantra (16 words)
- 🙏 Hare Krishna Half (8 words)  
- 💕 Radhe Radhe (1 word)
- ✏️ Custom mantras

## Installation

### iPhone/iPad
1. Open in Safari
2. Tap Share → "Add to Home Screen"

### Android
1. Open in Chrome
2. Tap Menu → "Install app"

### Desktop
1. Open in Chrome/Edge
2. Click install icon in address bar

## Usage

1. Select your mantra
2. (Optional) Calibrate microphone for Audio mode
3. Choose detection mode (Google/Audio/Tap)
4. Start chanting!

## Tech Stack

- Vanilla JavaScript
- Web Speech API
- Web Audio API
- Service Worker (PWA)
- localStorage
